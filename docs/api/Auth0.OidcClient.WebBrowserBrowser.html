WebBrowserBrowser(Func<Window>, Boolean)

Create a new instance of WebBrowserBrowser with a custom Window factory and optional flag to indicate if the window should be closed.

Declaration
public WebBrowserBrowser(Func<Window> windowFactory, bool shouldCloseWindow = true)
Parameters
Type Name Description
System.Func<System.Windows.Window> windowFactory

A function that returns a System.Windows.Window to be used for hosting the browser.
System.Boolean shouldCloseWindow

Whether the Window should be closed or not after completion.
Examples

This sample shows how to call the WebBrowserBrowser(Func<Window>, Boolean) constructor.

Window ReturnWindow()
{
    return window; // your WPF application window where you want the login to pop up
}
var browser = new WebBrowserBrowser(ReturnWindow, shouldCloseWindow: false); // specify false if you want the window to remain open
